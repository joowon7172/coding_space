## 1. `merge` 충돌 해결하기

`git merge conflict-1`로 병합을 시도하면 충돌 발생

- 오류 메시지와 `git status` 확인
- VS Code에서 해당 부분 확인



**당장 충돌 해결이 어려울 경우 아래 명령어로 `merge` 중단**

```
git merge --abort
```

이러면 merge가 중단이 되고 main으로 돌아가게 됨



해결 가능 시 충돌 부분을 수정한 뒤 `git add .`, `git commit`으로 병합 완료
